# Shareables

A simple Android sample project demonstrating how to capture a Jetpack Compose composable as an image and share it with other apps using Intent.

## Purpose

This project serves as a basic example for developers looking to:
- Capture a composable as an image (bitmap)
- Share the image via Android Intent to other apps

## Features

- Minimalistic single-activity structure
- Shows how to capture composable as an image
- Demonstrates passing the captured image to other apps
